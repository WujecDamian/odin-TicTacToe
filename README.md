# Tic-Tac-Toe

Live preview \/
[https://wujecdamian.github.io/odin-TicTacToe/]

Simple, stylish Tic-Tac-Toe game created with HTML, CSS and vanilla JavaScript.

Two players take turns placing their marks (circle / cross) on a 3×3 grid.  
First player to get three in a row (horizontally, vertically or diagonally) wins the round.  
Score is kept across multiple rounds.

## Features

- Classic 3×3 Tic-Tac-Toe gameplay
- Score tracking across rounds
- Visual indication whose turn it is (circle/cross icon)
- Win / Draw detection with temporary message
- "RESET SCORE" button to start fresh
- Nice hover effects and glowing icons
- Dark theme with neon accents

## Technologies

- HTML5
- CSS3 (modern properties: grid, filter, drop-shadow, aspect-ratio…)
- Vanilla JavaScript (no frameworks)

## How to Play

1. Open `index.html` in a modern browser
2. Enter name for **Player 1** (plays **O** / circle)
3. Enter name for **Player 2** (plays **X** / cross)
4. Click on any empty square to place your mark
5. First to get **3 in a row** wins the round
6. Score updates automatically
7. Use **RESET SCORE** button to set both scores to 0
